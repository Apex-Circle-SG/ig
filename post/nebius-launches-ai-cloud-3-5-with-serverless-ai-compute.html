Nebius Launches AI Cloud 3.5 with Serverless AI Compute

Introduction

Nebius has officially rolled out AI Cloud 3.5, a next generation platform that brings serverless AI compute to the forefront of cloud services. This launch marks a significant step forward for businesses that want to harness artificial intelligence without managing complex infrastructure. By abstracting away servers, Nebius enables developers to focus on model training, inference, and application logic while the platform automatically scales resources based on demand.

What Is Nebius AI Cloud 3.5?

AI Cloud 3.5 is the latest iteration of Nebius’s cloud offering, designed specifically for AI workloads. It combines high performance GPUs, optimized storage, and a fully managed serverless layer that executes functions in response to triggers such as API calls, data uploads, or scheduled events. The platform supports popular frameworks like TensorFlow, PyTorch, and Hugging Face, allowing teams to deploy models with minimal configuration.

Key Features of Serverless AI Compute

  • Instant scaling: Resources spin up in seconds and scale down to zero when idle.
  • Pay‑per‑use billing: Customers pay only for the compute time their functions consume.
  • Pre‑built AI containers: Ready‑to‑run images for common model serving tasks.
  • Native integration with Nebius data lakes and streaming services.
  • Security isolation: Each function runs in a hardened sandbox with encrypted data in transit and at rest.
  • API first design: Simple REST and GraphQL endpoints for triggering AI jobs.
  • Monitoring and logging: Built‑in dashboards provide real‑time metrics on latency, throughput, and cost.

Benefits for Developers and Enterprises

For developers, the serverless model eliminates the need to provision clusters, manage Kubernetes, or worry about patching GPU drivers. Teams can push code to a repository and have Nebius automatically build and deploy the corresponding function. Enterprises benefit from reduced operational overhead, predictable cost structures, and the ability to run experimental AI projects alongside production workloads without risking performance degradation.

Real‑World Use Cases

  • Real‑time image recognition for retail inventory systems.
  • Natural language processing bots that handle customer support queries.
  • Generative AI pipelines that create marketing copy on demand.
  • Financial fraud detection models that score transactions as they arrive.
  • Scientific simulations that burst compute for short, intensive training runs.

How Nebius AI Cloud 3.5 Stacks Up Against Competitors

Compared with traditional AI cloud offerings that require users to manage virtual machines or Kubernetes clusters, Nebius AI Cloud 3.5 delivers faster startup times and lower idle costs. While competitors such as AWS SageMaker Serverless Inferentia and Google Cloud AI Platform provide similar serverless capabilities, Nebius differentiates itself through a tighter integration with its own data storage layer and a pricing model that charges per second of GPU usage rather than per minute. Benchmarks show that a typical inference workload sees a 30% reduction in latency and a 40% decrease in monthly spend when moving from a comparable VM‑based setup to Nebius AI Cloud 3.5.

Getting Started with Nebius AI Cloud 3.5

  1. Sign up for a Nebius account and verify your email.
  2. Navigate to the AI Cloud console and create a new namespace.
  3. Choose a serverless function template (e.g., text classification, image generation).
  4. Upload your model artifacts or point to a repository hosted on Nebius Git.
  5. Configure the trigger – HTTP endpoint, queue message, or cron schedule.
  6. Deploy the function and monitor its performance via the built‑in analytics dashboard.
  7. Set up alerts for cost thresholds and latency targets.

Pricing and Availability

Nebius AI Cloud 3.5 is available in all major regions including North America, Europe, and Asia Pacific. Pricing starts at $0.0005 per GPU‑second for the baseline T4 instance, with discounted rates for reserved capacity and sustained usage. New customers receive a $200 credit to explore serverless AI compute during the first month. Detailed pricing tables can be found on the Nebius website under the AI Cloud 3.5 product page.

Future Roadmap

These include support for FPGA‑based accelerators, expanded model hub integrations, and advanced AutoML capabilities that automatically tune hyperparameters within the serverless environment. The company also plans to introduce multi‑region failover for mission‑critical AI services, ensuring high availability even during regional outages.

Conclusion

The launch of Nebius AI Cloud 3.5 with serverless AI compute represents a meaningful shift toward more accessible, cost‑effective, and scalable AI infrastructure. By removing the burden of server management, Nebius empowers both startups and large enterprises to innovate faster and bring intelligent applications to market with confidence. As AI continues to permeate every industry, platforms like AI Cloud 3.5 will play a pivotal role in shaping the next wave of digital transformation.

FAQ

What is serverless AI compute?

Serverless AI compute refers to a cloud execution model where the provider manages the underlying infrastructure, automatically scaling resources in response to demand, and charging only for the actual compute time used by AI functions.

Do I need to refactor my existing models to use Nebius AI Cloud 3.5?

In most cases, no. The platform accepts standard model formats such as SavedModel, TorchScript, and ONNX. You can deploy your existing models with minimal changes, primarily focusing on setting up the appropriate trigger and input/output handling.

How does Nebius ensure data privacy and security?

All data processed by AI Cloud 3.5 is encrypted in transit using TLS and at rest with AES‑256 encryption. Functions run in isolated containers, and Nebius complies with industry standards such as SOC 2, ISO 27001, and GDPR.

Can I combine serverless functions with traditional VM‑based workloads?

Yes. Nebius allows you to run serverless AI functions alongside virtual machines or Kubernetes clusters within the same project, giving you flexibility to choose the right tool for each workload.

What support options are available for AI Cloud 3.5 users?

Nebius provides 24/7 technical support via email, chat, and phone for paid plans. Community forums, documentation, and tutorial videos are also freely accessible to help users get started and troubleshoot issues.
