Swamp Workflow and Commands

Swamp uses several workflows and commands to help you bring your API models to life. Let’s look into a few key commands:

  • swamp repo init --tool claude: Before any operation, Swamp sets up a repository with necessary structures. For Claude/OpenClaw AI integration, the --tool claude flag is used.
  • swamp model create <type> <name>: Creates a new model definition. For example, swamp model create command/shell github-issues would set up a model for GitHub issues.
  • swamp model method run <model_name> <method_name>: This command executes a specific method on your model, helping you test it properly.
