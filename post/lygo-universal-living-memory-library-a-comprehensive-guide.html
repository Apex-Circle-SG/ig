What is the LYGO Universal Living Memory Library?

The LYGO Universal Living Memory Library is a universal upgrade skill designed for LYGO systems that establishes a minimal yet durable memory library framework. This sophisticated system operates as a pure advisor, meaning it performs no actions unless explicitly invoked by the user.

Core Architecture

At its foundation, the library implements a strict, low-noise memory index that maintains a maximum of 20 files in the active index. This constraint ensures a small context footprint while preserving essential information. The system employs {FRAGILE} tagging to flag files requiring manual review, creating a balanced approach between automation and human oversight.
